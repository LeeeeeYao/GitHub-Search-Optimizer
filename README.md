# GitHub-Search-Optimizer
Optimize your GitHub search queries with AI.

一款可以直接在浏览器插件中搜索并筛选Github上的项目，代码等。目前是一个初级版本，
后续会添加和你搜索的项目类似的项目联想，以及利用AI优化你的搜索关键词，让搜索结果更加符合你的预期。
